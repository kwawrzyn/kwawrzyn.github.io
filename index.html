Notable Projects

Python Pipelines for MRI Simulations

I used Python to codify algorithm chains and pipelines that help automate workflows within simulation platforms, such as Sim4Life & EMPro. Python was used to build sim models, manage sim sequences, and handle data processing. The sims are used to study the electromagnetic interactions and thermal responses of medical devices inside and around the MRI environment or as part of verification & validation procedures.

Outcomes:

Published ISMRM19: Abstract | Digital Poster

Programming a New MRI Pulse Sequence

I demonstrated the viability of a new MRI method by programmatically modifying, testing, troubleshooting and calibrating a novel MRI sequence for lung imaging. I setup equipment and operated an MRI scanner to collect raw data. MATLAB was used to codify the data analysis routines. Data was processed in both the time and frequency domains. Improvements to SNR and visual appearance of images was accomplished through data cleaning. Results were quantified using Fourier analysis and statistics software such as Prism GraphPad.

A Protocol for Imaging Electrokinetics using OCT

I developed a novel protocol showing the feasibility of optical imagery to detect the electro-kinetic response in biological tissues. I prepared optical & electric equipment arrangement for data acquisition, collected raw data from an optical coherence tomography (OCT) system, and processed data using MATLAB programming. Fourier analysis methods were applied to quantify the results and produce enhanced images.

A Python Temperature System Logger

I used Python to develop this GUI application to interface and control a Temperature System Conditioner. The model-view-controller software design pattern was utilized with object oriented programming to separate each element into its own class file. I used Python's TKinter library for the visuals and I created my own library of methods with the PySerial module for the serial RS-232 communication. The application connects to the hardware, logs temporal temperature data into a text file, and visually displays a live temperature plot during data collection.
